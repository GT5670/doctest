= Post-installation integrations

After installing {ProductShortName}, there are a several scenarios that require you to complete some additional work, to ensure {ProductShortName} functions properly.

If you integrated your own instance of Quay into {ProductShortName}, or if you want to use private repositories in Quay, then you must now integrate Quay into ACS. This ensures ACS has access to the repositories you use in Quay.  

Additionally, if you integrated Jenkins into {ProductShortName}, you must configure Jenkins, using the Jenkins UI ensures that Jenkins can run the build pipelines that {ProductShortName} requests. 

Additionally, if you integrated GitLab  into {ProductShortName}, you must configure GitLab, using the GitLab UI ,Webhooks for automated pipeline triggers.

== (Optional) Configuring GitLab Webhooks for automated pipeline triggers

Set up webhooks and secrets in GitLab to automatically trigger pipeline run in {RHDHShortName} upon code updates.
