`input` is a rego object that contains all of the information about your container image, its signature, and its attestations. `input.attestations` refers to the `attestations` attribute of the `input` object which contains a list of attestations associated with the image. See link:https://enterprisecontract.dev/docs/ec-cli/main/policy_input.html[Policy Input] for more information about where and how {ECShort} defines `input.attestations` contents.
